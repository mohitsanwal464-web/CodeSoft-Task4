# Task 4 - Sales Data Analysis Using Python

## Objective
Analyze customer sales data using Python and visualize the results with charts.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset
The dataset contains:
- Customer ID
- Customer Name
- Age
- Location
- Product
- Purchase Amount

## Steps Performed
1. Imported the dataset using Pandas.
2. Displayed the first few rows of the dataset.
3. Identified top customers based on purchase amount.
4. Calculated location-wise total purchase.
5. Calculated product-wise total purchase.
6. Created a Bar Chart for location-wise purchase.
7. Created a Line Chart for product-wise purchase.

## Output
- Top Customers List
- Location-wise Purchase Summary
- Product-wise Purchase Summary
- Bar Chart (Location Wise Purchase)
- Line Chart (Product Wise Purchase)

## Conclusion
This project demonstrates how Python can be used for data analysis and visualization. Pandas was used for data processing, and Matplotlib was used to create charts for better insights.
